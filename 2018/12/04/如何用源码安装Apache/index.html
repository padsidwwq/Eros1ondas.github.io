如何用源码安装Apache

[TOC]

一、在官网下载最新版的Apache

1.进入官方下载地址http://httpd.apache.org/download.cgi 。

2.找到最新的源码包，并下载。

二、下载Termius，用它来连接VMware里的Centos虚拟机。

1.下载打开Termius，打开之后是这样子的。

2.点击创建主机

3.设置主机信息

4.双击连接刚刚添加的主机

5.点击yes

6.连接完成

三、下载Cyberduck以远程上传文件

1.下载Cyberduck，并打开，如图。

2.点击新建连接

3.设置要连接的主机信息

4.选择允许

5.连接成功

6.选择要上传的文档，并拖拽到目标文件夹

四、安装Apache

1.进入Termius，连接centos7

2.执行yum groupinstall “Development Tools“ -y

3.找到apache的压缩包，执行tar -zxvf httpd.

4.解压之后会多处来一个httpd.2.4.73的文件，cd进去

5../configure来配置httpd

6.这时会报错，提示缺少依赖包，我们yum search apr 搜索依赖包apr

7.yum install apr-devel apr-util-devel apr-util apr 来安装四个依赖包 apr-devel apr-util-devel apr-util apr 此过程有询问的话输入y（yes）

8.重新执行./configure

9.这时又会报错，提示缺少依赖包，我们yum search pcre来搜索依赖包

10.yum install pcre-devel pcre-static pcre-tools pcre 来安装四个依赖包 此过程有询问输入y（yes）

11.重新运行./configure 显示完成了

12.编译运行 make && make install，然后完成

13.尝试执行/usr/local/apache2/bin/apachectl start来启动服务

14.报错了，修改配置文件。因为apache默认是注释掉了ServerName，我们要去掉注释，配置文件在

/usr/local/apache2/conf/httpd.conf,把Server Name那项的井号删掉

15.重新启动恢复正常。

五、导入网站

1.找到Apache默认主页并删除 rm -rf /usr/local/apache2/htdocs/index.html

2.找到自己的GitHub网站文件夹使用Cyberduck上传至Centos 7 的/usr/local/apache2/htdocs

3.关闭Centos7的防火墙输入命令 systemctl stop firewalld

4.浏览器输入localhost:80或者本机IP:80来浏览自己的博客

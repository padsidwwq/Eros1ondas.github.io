制作我的博客系统

2018-12-03  

如何开通自己的博客
一、在GitHub网站上注册一个账号并进行相关的设置：
1.1输入https://github.com，就是下面的这个界面

1.2点击注册按钮

1.3填写用户名、邮箱、密码等信息，然后完成那个验证点击创建账户

1.4选择一个免费版

1.5随便选信息

1.6

1.7 提示你去邮箱里点击收到的邮件以激活账户
忘了截图
2 创建一个库 第一处填写和注册名一样的库名然后后面跟上.github.io一定要以io结尾。
第二处写介绍 随便写 第三处选择开放的

2.1完成之后就会出现如下界面啦

2.2点击红线处以设置

3.在settings找到自己的url，进入settings之后一直往下找到自己的地址

3.1浏览器访问

也可以点击新建文件来自定义网页，文件名随便写

随便写内容，然后点击最底部的保存

完成

二、下载安装一些工具
2.1安装vscode 双击运行安装包之前都是默认下一步，到这一步全选，然后下一步直至完成

2.2安装git
直至选择编译器时选择vscode，一直下一步，忘了截图
2.3 安装GitHub desktop 会让你安装.Net 装完要重启
2.4安装node.js一直下一步
三、设置网站
3.1纯英文路径下创建一个文件夹blog（名字随意）

3.2进入blog文件夹在搜索栏输入cmd以打开命令行

3.3输入命令npm install -g hexo-cli然后回车 必须有网

成功了的样子

3.4再键入命令hexo init . 会给你创建很多文件夹

3.5 返回看到blog文件夹，右键blog文件夹以vscode打开

找到blog–>_config.yml

修改选项 语言处写ZH-CN
在url处写上自己的GitHub给你的url

3.6 然后运行命令在blog里运行cmd输入hexo generate

3.7输入 hexo server启动服务 启动之后点击运行访问，浏览器输入他给你的url

四、配置导入到GitHub
4.1打开GitHub desktop，登录自己的GitHub账户
4.2出现这一步点继续然后完成，就会出现下图

4.3点击克隆库，也就是上图最右边一个选项
4.4点击自己的url，选择克隆

4.5完成之后会在你的计算机–>文档里创建一个GitHub文件夹–>自己的站点 把里面的内容删了
4.6把blog的public文件夹打开，复制里面的内容到文档的自己的站点的文件夹下面，如下图

4.7打开GitHub desktop ，它会自动检测到文件的变动，这时你自己在那起个名字（比如我的第一个博客样式）然年点击提交
4.8提交完之后点击
4.9刷新自己的博客网页
完成

2用YUM安装LAMP环境

LAMP=Linux+Apache+Mysql+PHP

一.Linux

1.准备centos 6，配置好IP

二、用yum安装Apache

1.输入命令yum install httpd httpd-devel httpd-manual mod_ssl mod_perl mod_auth_mysql -y

2.防火墙打开TCP80端口，用vi编辑器打开/etc/sysconfig/iptables,输入-A INPUT -m state –state NEW -m tcp -p tcp –dport 80 -j ACCEPT来开启80端口

3.重启防火墙 输入 service iptables restart

4.开启apache服务 service httpd start

5.打开浏览器输入本机IP测试httpd服务

6.将apache开机自启

三、yum安装Mysql

1.yum安装Mysql，输入命令 yum install mysql mysql-server mysql-devel -y

2.成功

3.启动MySQL服务，输入service mysqld start。他会初始化，所以第一次启动得等一下。

4.对MySQL进行安全设置mysql_secure_installation，下图第一步点回车

5.点y(yes)对root（数据库的管理员）设置密码

6.输入密码

7.删除匿名账户，点yes

8.点yes,一路yes

9.登录mysql 输入命令mysql -uroot -p

10.完成啦

11.将mysql开机自启，输入命令chkconfig mysqld on

四、用yum安装php

1.安装php 输入命令yum install php php-mysql php-common php-gd php-mbstring php-mcrypt php-devel php-xml -y

2.验证一下php,在/var/www/html/下新建一个index.php的文件，并写入<? php phpinfo(); ?>

浏览器输入验证，成功。

扩展：使用wordpress在Linux中建设一个动态网站 wordpress是一个建设博客的网站

1.第一步使用Cyberduck将wordpress上传到centos 6.5到/var/www/html/

2.解压到此文件夹

3.将wordpress下的文件都复制到html目录下 cp -r(因为有文件夹) ./* ../

4.给html加权限 chown -R (给文件夹下的包括文件夹更改权限) apache:apache ./

5.浏览器输入localhost:80来验证

6.在MySQL里创建一个新库 create database 库名

7.对网站进行设置，第一项为刚刚创建的库名

10.登录

11.完成

扩展2.使用VAuditDemo搭建一个漏洞测试网站

1.删除/var/www/html/下所有的东西。

2.将VAuditDemo上传到Linux中

3.将VAuditDemo解压。 unzip VAuditDemo.zip

4.将VAuditDemo下的VAuditDemo_Debug下的所有文件复制到/var/www/html/下

5.重启apache，service httpd restart

6.进入网站的配置 更改用户名密码和数据库等信息。vi /var/www/html/sys/conf.php

7.将html文件夹提权chown -R apache:apache ./

8.将网站配置文件导入到数据库，首先进入到install目录，在此目录中进入mysql， mysql -uroot -p

9.进入Ning那个库

10.输入命令 source install.sql；

12.完成
